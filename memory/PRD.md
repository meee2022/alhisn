# الحصن (Al-Hisn) - SafeLink Guard

## نظرة عامة
تطبيق فحص الروابط المشبوهة للمستخدمين العرب - يساعد في الكشف عن روابط التصيد والبرمجيات الخبيثة.

## التقنيات المستخدمة
- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS + Custom Green Cybersecurity Theme
- **UI Components**: Radix UI (shadcn/ui)
- **Animations**: Framer Motion + CSS Animations
- **Routing**: React Router v6
- **Forms**: React Hook Form + Zod

## الميزات المُنفذة
1. **الصفحة الرئيسية (Home)**
   - درع متوهج أخضر يعرض حالة الأمان
   - زر "فحص رابط" رئيسي
   - زر "مسح باركود" QR
   - إحصائيات الأمان اليومية
   - تنبيه أمني أخير

2. **صفحة فحص الرابط (URL Scanner)**
   - تصميم رادار متحرك
   - حقل إدخال URL مع زر لصق
   - زر "افحص الرابط الآن"
   - حالة "جاهز للفحص" مع أنيميشن

3. **نتائج الفحص (Scan Results)**
   - عرض حالة الرابط (آمن/مشبوه/خطير)
   - تفاصيل التهديدات
   - نقاط السمعة
   - كشف التصيد والبرمجيات الخبيثة وSSL

4. **سجل الفحص (History)**
   - تتبع جميع الروابط المفحوصة

5. **الخزنة الآمنة (Vault)**
   - حفظ الروابط الموثوقة

6. **نصائح أمنية (Tips)**
   - محتوى تعليمي بالعربية والإنجليزية

7. **الإعدادات (Settings)**
   - دعم اللغة العربية (RTL) والإنجليزية (LTR)

## التصميم
- **نظام ألوان**: أخضر سايبري داكن (#00ff64 على خلفية #040a04)
- **تأثيرات**: Glassmorphism + Neon Glow
